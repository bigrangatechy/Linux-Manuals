## How to Find More Equivalents

If an app you need isn't listed here:

1. **Search the App Center** (press Super, type "App Center") — search by function or name
2. **Search Flathub** at [flathub.org](https://flathub.org)
3. **Search Snap Store** at [snapcraft.io/store](https://snapcraft.io/store)
4. **Check alternativeto.net** — a website that lists alternatives for any app, filterable by Linux platform
5. **Search Ask Ubuntu** — "equivalent to [app name]" usually returns results
6. **Try Wine** — check [winehq.org](https://www.winehq.org) to see if the Windows app runs through Wine
7. **Use a web app** — many services work perfectly in a browser (Google Docs, Figma, Notion, YouTube Music)

> 💡 **Native Is Almost Always Better**
>
> A native Linux app will be faster, more integrated, and more stable than running a Windows app through Wine. Before trying Wine, check whether a Linux-native equivalent exists using the resources above. Wine is a fallback, not a first choice.

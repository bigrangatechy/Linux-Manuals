# Chapter 9.5: KDE Plasma 6.6 Desktop Tour (Kubuntu)

## Welcome to Kubuntu's Desktop

If you're reading this chapter, you're running **Kubuntu** with **KDE Plasma 6.6** — a completely different desktop experience from standard Ubuntu's GNOME 50, but built on the exact same underlying system.

This chapter mirrors the structure of Chapter 9, but tailored to KDE Plasma. If you're on standard Ubuntu, skip this chapter and continue to Chapter 10.

> 💡 **Quick Tip: Press the Meta/Windows Key**
>
> Throughout this chapter, you'll see references to the **Meta key** or **Windows key**. On most keyboards, this is the key with the Windows logo between Ctrl and Alt. Pressing it opens the application launcher menu by default.

## The Three Main Areas of Your Screen

Your KDE Plasma desktop can be divided into three main zones:

| Zone | Location | What It Contains |
|---|---|---|
| **Panel (Taskbar)** | Bottom edge (by default) | Application launcher, task manager, system tray, clock |
| **Desktop Area** | Center | Wallpaper, widgets, optional file icons |
| **System Tray** | Bottom-right corner | Notification area, network, audio, power, settings |

Unlike GNOME, which emphasizes keyboard navigation and minimalism, KDE Plasma presents a traditional, customizable interface reminiscent of Windows.

### The Panel (Taskbar)

By default, KDE Plasma uses a horizontal panel at the bottom of your screen. This is analogous to the Windows taskbar.

**Left Section:**

1. **Application Launcher Button** — Typically shows the Kubuntu penguin logo. Clicking opens the Application Menu (equivalent to Start Menu).
2. **Recent Documents** (optional widget) — Quickly access recently used files
3. **Task Manager** — Pinned applications on the left, running apps in the middle
4. **Add Widgets** — Quick access to add new panels or desklets

**Center Section:**

- Open windows appear as grouped buttons here
- Right-click to show window list, arrange by app, or adjust grouping behavior
- Hover over an icon to preview thumbnails of open windows

**Right Section (System Tray):**

A cluster of small icons providing quick access to:

1. **Network/Wi-Fi Icon** — Shows connection status, connects/disconnects networks
2. **Volume/Speaker Icon** — Adjust audio volume, select output devices
3. **Power/Battery Icon** (laptops) — Show battery level, power profiles
4. **Clipboard History** — Access clipboard history if enabled
5. **Notifications** — Bell icon showing recent alerts
6. **Digital Clock** — Date and time; hover or click for calendar
7. **Settings Gear** — Quick access to System Settings

Clicking any of these icons expands a dropdown panel with detailed controls. Right-clicking the panel itself gives you options to configure panels, add widgets, or lock them in place.

> 💡 **Customize Your Panel**
>
> Right-click anywhere on the panel → **Edit Panel** → Modify layout, add/remove widgets, change position. You're not limited to one panel — you can have multiple stacked panels, side bars, or floating docks.

### Adding Panels and Widgets

KDE Plasma's defining strength is customization. To customize:

1. **Enter Edit Mode**: Right-click desktop → **Enter Edit Mode**
2. **Add Panel**: Click "Add Panel" button or drag existing panels
3. **Add Widgets**: Click "Add Widgets" button, browse categories
4. **Configure Widgets**: Drag widget to edit size, right-click → Configure for options

Common default widgets on fresh Kubuntu installation:

- **Application Launcher** — Traditional start menu alternative
- **Folder View** — Display folders directly on desktop
- **Digital Clock** — Always visible in panel
- **Hard Disk Health** — Storage monitoring
- **Battery Monitor** — Laptop power status
- **Notes** — Sticky notes for reminders
- **Weather Report** — Current conditions and forecast

Widgets can live on panels, the desktop, or as floating "plasmoids." This flexibility dwarfs what GNOME offers.

> ⚠️ **Avoid Overwhelming Yourself**
>
> KDE gives you nearly infinite customization. Don't try to tune every setting on day one. Use the defaults, then gradually personalize based on what annoys you or what workflow gaps emerge. We cover deep customization in Chapter 26.

### The Application Launcher (Kickoff Menu)

The heart of KDE is its **Application Launcher**, often called **Kickoff**. By default, it sits on the far left of the bottom panel.

**Opening It:**

1. Click the **Penguin Logo** button
2. Or press the **Meta/Windows key**
3. Or use **Alt+F1** as a fallback shortcut

**Structure:**

| Tab | Contents |
|---|---|
| **Applications** | All installed software organized by category (Office, Graphics, Internet, etc.) |
| **Favorites** | Frequently-used apps (customizable) |
| **Recent Files** | Recently opened documents |
| **Places** | Quick links to Home, Documents, Network, Trash |
| **Computer** | Direct access to filesystem locations, drives, mounted volumes |
| **Run Command** | Execute commands without opening terminal (Alt+F2) |
| **Search Box** | Type to filter apps, files, settings, calculations |

**Efficient Usage:**

- **Keyboard Navigation**: Arrow keys navigate menus; Enter selects
- **Direct Search**: Start typing immediately after pressing Meta key
- **Pin Favorites**: Right-click any app in the launcher → **Add to Favorites**
- **Recent Items**: Keep workspace efficient by pinning daily essentials

> 💡 **Alternative Launchers**
>
> KDE offers several launcher styles beyond Kickoff:
>
> - **Application Dashboard** — Full-screen app grid (like macOS Launchpad)
> - **Application Menu** — Classic cascading menus (Windows XP style)
> - **Classic Menu** — Minimal two-column layout
>
> Switch launcher types: Right-click Application Launcher → **Alternatives** → Choose preferred style.

### Desktop Icons vs. Empty Space

Unlike GNOME, KDE Plasma includes **desktop icons by default**. The desktop isn't empty wallpaper — it contains shortcuts, folders, and mounted drive entries.

**What You'll Find Initially:**

- **Home Folder** — Your personal directory
- **Mounted Drives** — External storage appears as clickable icons
- **Recycle Bin** — Deleted files pending permanent removal
- **User Folders** — Optional shortcuts to Documents, Pictures, Downloads

**Managing Desktop Items:**

- **Drag/Drop** — Reorder icons freely
- **Align Grid** — Right-click desktop → Align Icons → Grid (or Free Position)
- **Hide Temporarily** — Right-click desktop → View Options → Hide Desktop Icons
- **Sort Automatically** — Name, size, type, date modified

This familiarity makes KDE comfortable for Windows refugees who miss having direct access to frequently-used items.

## Window Management Basics

KDE manages windows similarly to Windows/macOS but with additional features baked in.

### Moving Windows

- **Drag title bar** — Move anywhere on screen
- **Double-click title bar** — Minimize or maximize based on action
- **Middle-click title bar** — Roll up/down window to just title bar

### Resizing Windows

- **Drag edges/corners** — Standard resize behavior
- **Corner snapping** — Some configurations support corner docking
- **Tile manually** — Drag to edges (configurable behavior)

### Tiling and Window Placement

KDE has robust window placement automation:

1. **Smart Placement** — Avoid overlapping existing windows
2. **Rigid Placement** — No overlap regardless of current load
3. **Under Mouse Pointer** — Place where cursor lands
4. **Cascade New Windows** — Offset each new window slightly

To configure:
```bash
kcmshell5 kwinoptions  # Opens Window Rules module
# Or navigate via System Settings → Workspace Behavior → Window Management

Snap Layouts (Improved in Plasma 6.6)

Plasma 6.6 introduces enhanced tiling similar to Windows Snap:

    Drag to Edge — Pulled windows snap to halves like Windows
    Drag to Corners — Snaps to quarters (each quadrant fills independently)
    Shortcut Override — Use Super+Arrow Keys to tile rapidly
    Resizable Dividers — Split ratio adjustable after tiling

This feature works identically to Windows' Snap for migrants seeking familiar behavior.
Maximizing/Restoring Windows

    Maximize: Double-click title bar or Alt+F10
    Minimize: Click minimize button or Alt+F9
    Close: Alt+F4 or Ctrl+W depending on context

Fullscreen Apps

Press F11 or use app's View menu to toggle fullscreen mode. Exit with Esc or F11.
Workspaces and Virtual Desktops

Like GNOME, KDE supports virtual desktops — but implements them differently.
Managing Multiple Workspaces

    Default Count: Four workspaces available by default
    Navigate Using Desktop Pager Widget — Visible on panel by default
    Keyboard Shortcuts:
        Ctrl+Alt+←/→ — Switch between workspaces horizontally
        Ctrl+Alt+↑/↓ — Navigate vertically (if configured)
    Mouse Gestures — Touchpad swipes switch workspaces if enabled
    Move Windows Between Workspaces — Ctrl+Shift+Win+Number

Configuring Workspaces

Open System Settings → Workspace Behavior → Virtual Desktops:

    Increase/Decrease Workspace Count — Set fixed number or dynamic
    Show Desktop Pager in Panel — Enable/disable visual representation
    Animate Transitions — Fade/swipe effects when switching
    Per-Application Desktop Assignment — Assign specific apps to specific workspaces automatically

Workspaces help separate workflows logically:

    Workspace 1: Work applications (email, documents, terminal)
    Workspace 2: Entertainment (browser, media player, games)
    Workspace 3: Creative tools (image editors, video software)
    Workspace 4: Development environments (code editors, debuggers, documentation)

System Notifications

Notifications slide in from the top-right corner, remaining visible until dismissed or expired.

Clicking the notification bell icon in the system tray reveals notification history.
Managing Permissions

Review and control notification sources:

    Open System Settings → Notifications
    Browse list of requesting applications
    Toggle Allow/Block per app
    Define silent hours (similar to Do Not Disturb)

Advanced option: Notification Filter Rule Wizard to create complex suppression rules (e.g., silence specific apps during certain hours).

    💡 Focus Modes

    Plasma 6.6 introduces Focus Modes allowing pre-defined quiet periods triggered manually or by location/time schedules.

Power Management (Critical for Laptops)

Kubuntu excels at power efficiency through granular control:

Open System Settings → Power Management:
General Settings

    Energy Saving Profiles: Balanced, Performance, Power Saver
    Lid Close Action: Sleep, Hibernate, Shut Down, Prompt
    Power Button Action: Adjustable trigger duration
    Auto Suspend Delay: Time before automatic sleep on battery/plugged-in

Display Specific

    Dim Brightness After — Automatic screen dimming threshold
    Turn Off After — Screen timeout period
    Reduce Backlight When Dimmed — Additional brightness reduction

Hard Drive & CPU Specific

    Idle Suspension — Put HDD into standby when idle
    CPU Governor — Balance performance vs energy consumption

All settings apply per power profile — create customized profiles for scenarios (meeting mode, presentation mode, gaming mode).
Online Accounts Integration

Sync cloud services across KDE applications:

    Open System Settings → Online Accounts
    Sign in to supported providers:
        Google
        Nextcloud
        OwnCloud
        Microsoft Exchange ActiveSync
        Twitter/X (partial support)

Data synchronizes into corresponding KDE apps:

    Calendar events into Kalarm/KOrganizer
    Contacts into Address Book
    Cloud files into Dolphin file manager sidebar
    Messages into mail clients (KMail, Thunderbird)

Google integration may require granting OAuth permissions in browser during setup.

    ⚠️ Security Note

    OAuth tokens persist locally unless revoked. Regularly review connected accounts in Settings → Online Accounts and disconnect unused ones.

The File Manager: Dolphin

KDE ships with Dolphin, widely regarded as the best Linux file manager. It packs more features than Nautilus (GNOME Files) while remaining responsive.
Opening Dolphin

    Click folder icon in panel
    Meta+E keyboard shortcut
    Navigate to Home folder from Application Launcher

Core Features
Feature	Benefit
Dual-Pane Mode	Press F3 — copy/move files between panes effortlessly
Split View	Divide single pane vertically for nested navigation
Breadcrumb Navigation	Address bar with full path display
Preview Thumbnails	Inline image/document previews for instant identification
Column View	Hierarchical folder tree view option
Remote Connection Sidebar	FTP/SFTP/WebDAV mounts integrated natively
Terminal Emulator Panel	Press F4 — embedded terminal docked at bottom
Tabs Support

Dolphin handles tabs aggressively:

    Ctrl+T — Open new tab
    Ctrl+W — Close current tab
    Ctrl+Tab — Cycle between tabs
    Ctrl+PgUp/PgDn — Navigate tab order
    Drag Tab — Float detached windows for split-view comparison

Tabs make multi-folder workflows significantly more efficient compared to single-window file managers.
Advanced Features

Bookmarks and Places Bar:

    Pin frequently-accessed directories to sidebar
    Right-click folder → Add to Places
    Reorder and organize bookmarks visually

File Operations:

    Copy (Ctrl+C) + Paste (Ctrl+V)
    Cut (Ctrl+X) + Paste (Ctrl+V)
    Rename (F2) — Inline rename without dialogs
    Delete (Del) moves to trash
    Shift+Delete — Permanent deletion bypassing trash
    Undo Delete (Ctrl+Z) — Recover accidentally deleted items

Integrated Search:

Type keywords into search field — instant filename matching. Add filters:

    File type
    Size constraints
    Modified date range
    Content text search (requires indexing)

Dolphin indexes commonly-used folders transparently. Full filesystem search available via Locate backend.

Troubleshooting Common KDE Issues

Even polished interfaces face occasional hiccups:
Session Corruption Fix

Corrupted session cache prevents proper restoration:

rm -rf ~/.cache/plasma-sessions/*

Log out/in afterward.
Plasma Shell Crash Recovery

If the desktop disappears unexpectedly:

kstart5 plasmashell || plasma5support --session

Or reboot entirely.
Missing Icons in System Tray

Restart system tray service:

killall plasmashell && kstart5 plasmashell

Slow Startup Delays

Optimize autostart entries:

    Open Autostart application
    Review and disable unnecessary launch programs
    Increase delay timers for heavy services

Wayland/X11 Session Confusion

On login screen, click gear icon and select:

    Plasma (Wayland) — Preferred modern session
    Plasma (X11) — Legacy compatibility mode

Switch sessions after successful logout.
